# Open Trivia App

## Live demo
https://open-trivia-frontend.onrender.com

## Lokaal draaien

### Backend draaien
cd open_trivia_backend

#### Voor Windows Command Prompt
gradlew bootRun

#### Voor Linux/macOS / PowerShell / Docker
./gradlew bootRun

### Frontend draaien
cd open_trivia_frontend 
npm install
npm start
