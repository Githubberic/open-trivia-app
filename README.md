# Open Trivia App

## Live demo
https://open-trivia-frontend.onrender.com

>> Let op: de eerste keer dat je de pagina bezoekt kan het ongeveer 30-60 seconden duren voordat de vragen geladen zijn.
>> De backend draait op de free tier van Render en slaapt na 15 minuten inacitiviteit. De backend moet vervolgens weer opstarten.

## Lokaal draaien

### Backend draaien
```
cd open_trivia_backend
```

#### Voor Windows Command Prompt
```
gradlew bootRun
```

#### Voor Linux/macOS / PowerShell / Docker
```
./gradlew bootRun
```

### Frontend draaien
```
cd open_trivia_frontend 
npm install
npm start
```
